# Checkbox

With `MDX`, we can define a story for `Checkbox` right in the middle of our
Markdown documentation.

<Canvas>
  <Story 
    name="Unchecked"
    args={{ 
      label: 'Unchecked',
    }}
    render={Checkbox} />
  <Story 
    name="Checked"
    args={{ 
      label: 'Unchecked', 
      checked: true,
    }}
    render={Checkbox} />
  <Story 
    name="Secondary" 
    args={{
      label: 'Secondary', 
      checked: true, 
      appearance: 'secondary',
    }}
